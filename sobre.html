Sobre mim

Beeeem vindo(a) novamente, se você chegou até aqui sinto que se interessou pelo meu conteúdo hahaha!! Nessa parte do site vou dedicar pra contar um pouco da minha história, espero que você goste :)

Me chamo Guilherme Marangão Calil, nascido em 09/06/2004 (18 anos) criado na cidade de Varginha, um município mineiro localizado mais ao sul do Estado. Desde pequeno sempre fui interessado pela tecnologia, ao longo da vida fui me apaixonando mais e mais na área. Fui pesquisando sobre o emprego e assim decidi que queria isso pra minha vida. Hoje além da faculdade faço diversos cursos, procurando sempre evoluir, além disso, realizo vários testes sobre programação dentre eles, criação de sites... Nos dias atuais faço Estágio em TI e tem sido uma experiência maravilhosa, trabalho ao lado de uma equipe maravilhosa e que me ensinam muito sobre diversos assuntos. Nesses quase 3 anos estuando programação pude notar no quão extensa essa área é, como é cheia de oportunidades para quem deseja embarcar nessa jornada...

Conhecimentos

HTML5

Conhecimento profissional para trazer todo seu site, utilizamos tags semâticas para dar mais significados e além disso, a lógica está sempre presente para deixar seu site ainda mais projetado!!

CSS3

CSS, a principal ferramenta para a estilização de seu site, profissionalismo na escritura dos códigos para deixa-los o mais limpo para assim ter uma velocidade de um foguete :0, tratamos a performace de seu site com profissionalismo!!!

Node.Js

Ainda nos estudos, porém já com um bom conhecimento sobre a linguagem, utilizado bastante para excutar códigos JavaScript fora de um navegador Web, além de ser um software de código aberto...

JavaScript

JavaScript, linguagem usada tanto para o Front-End quanto para o Back-End, linguagem bastante completa para trazer tudo que sua Página da Web precisa, conhecimento avançado :)

React.Js

Também ainda nos estudos, porém já com conhecimentos, React é uma biblioteca Front-End JavaScript de código aberto com foco em criar interfaces de usuários em páginas web!!!

Git e GitHub

Git é um software VCS local que permite desenvolvedores salvar snapshots de seus projetos ao longo do tempo. Já GitHub é uma plataforma baseada na web que incorpora os recursos de controle de versões do gitHub para que possam ser usados colaborativamente...
